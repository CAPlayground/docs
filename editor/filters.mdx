title: "Filters"
description: "Front-end Filters tab for per-layer post-processing."
---

Filters let you apply post-processing effects to any layer from the Inspector.

## Filters Tab

- **Add filter**
  Opens a dropdown to add a new filter. Multiple filters and duplicates are supported.

- **Enable/disable**
  Toggle a filter on or off without removing it.

- **Remove**
  Deletes the selected filter from the layer.

- **Value**
  Numeric control whose label depends on the filter type.

## Supported Filters

These match the UI options and their value labels:

- Gaussian Blur — Radius
- Contrast — Amount
- Hue Rotate — Angle
- Invert
- Saturate — Amount
- Sepia — Intensity

## Where to find it

- Select any layer
- Open the Filters tab in the Inspector
- Use the Add filter dropdown to insert effects

## Notes

- Filters are applied in the order they appear.
- You can add the same filter multiple times with different values.

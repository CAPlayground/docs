CAPlayground is the successor to [OpenPoster](https://openposter.github.io).

CAPlayground is a powerful web interface that allows you to create beautiful animated wallpapers for iOS and iPadOS on any desktop computer.

## Usage

Open the web interface at https://caplayground.pages.dev.

Once on the website, you can either log in or create an account.

<Info>
  For more information on accounts, see [Accounts](/account)
</Info>

Once authenticated, you can create a project on the [Dashboard](/account/dashboard).
